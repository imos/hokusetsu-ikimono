成虫の見られる場所

・

山地の林の周辺の植物がはえている環境や堤防の上の草がはえているような環境

鳴き声

・

ジーという連続音

北摂地域の都市に近い低地では、堤防などの雑草がはえている場所で比較的よく見られます。また、山地の周辺部の草のはえている場所でもよく見られます。

夜に観察をしていると、エノコログサなどの種子を食べているところもよく見かけます。

(大阪府池田市周辺を基準にしています)

2010年8月22日…兵庫県川西市山下

　クサキリは地面も歩いていることが多いのですが、上の写真のようにイネ科の植物で種子を食べている場面にもよく出会います。

2011年7月28日…大阪府池田市伏尾町

　木々をかき分けて進んでいると足下にクサキリが止まっていました。あまり7月にクサキリを見た記憶がありません。もうこの時期には成虫になっていました。

2012年8月26日…兵庫県川西市山下

　観察会をしているとクサキリを見つけた人がいました。写真はメスですがいっしょに♂もいました。
