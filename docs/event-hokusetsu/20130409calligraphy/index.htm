小堀奈穂カリグラフィー作品展

　

　

　

　カリグラフィーという言葉になじみのない方もおられる方と思います。西洋書道、ちょっと無理やり日本語にした感じがしますが。上の写真のようなカードを見たことがあることでしょう。そこに書かれているのがカリグラフィーです。業者の印刷と思ってしまいますが練習をすれば自分で書くことができるのです。お気に入りのカードにこのような文字を入れられるなんて素敵ですね。

　写真のカードを書かれたのは小堀奈穂先生です。定規を使ったりしながら書きますが、小堀先生は何も使わずに書かれます。驚きでした。その小堀先生の作品展です。

2013年4月9日(火)～4月26日(金)
月～金　AM11:00～PM7:00
土・日　　AM10:00～PM5:00

【作家略歴】
小堀奈穂(ヨーロピアンカリグラファー)
大阪を中心に活動するカリグラファー。
カリグラフィー歴は20年近くに及ぶ。
日本の文化を解し、西洋の文化に携わり、
広い視野を備えるべく、探究心を持つ。
指導歴8年。博物館学芸員。
