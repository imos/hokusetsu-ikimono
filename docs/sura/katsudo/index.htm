スラ人と自然の多様性ネットワーク当面の活動(2018年、平成30年)(案)

 

1.    川西市教育委員会等が行う自然体験学習、教員研修で生物多様性の意味や重要性の浸透を図る研修等を行う。

2.    地域により多くの環境カウンセラー(環境省)の育成を目指す。

3.    地域や地球規模の環境に関わる内容、持続可能な社会等EGSに関する学習会の実施、参加を行う。

4.    安定した会の運営を目指す。

5.    より具体的な活動は会議で企画・立案する。
