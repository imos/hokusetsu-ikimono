ナマズ目(ギギ科)

ギギ

　ナマズに似たギギ科の魚です。口ひげが4対で8本あります。夜行性です。「ギーギー」という音を出すのが名前の由来です。北摂地域では、「ギギタ」ともよんでいます。

(大阪府池田市周辺を基準にしています)

2010年10月5日…兵庫県川西市小花産(飼育)

　上の写真は1年前に採集したものを飼育しているものです。ギギは肉食ですが、金魚の餌などを与えてもずっと飼育することができます。

2010年8月19日…兵庫県川西市小花

　夏に水辺で川の生き物の観察をしているとかわいい小さなギギが採集されます。

2006年6月21日…大阪府池田市秦野(飼育)

　水槽で飼育されていたギギです。他の魚たちといっしょに飼われていました。口ひげのようすがよく分かります。
