コイ目(ドジョウ科)

ドジョウ

2011年4月29日…大阪府池田市

　ドジョウは猪名川の本流で見られる魚ではありません。周囲の田や池で見られます。川で見られるのは、シマドジョウとスジシマドジョウです。もともとはどこにでも見られたドジョウですが、最近は見られる場所が限られてしまいました。

(大阪府池田市周辺を基準にしています)

2011年4月29日…大阪府池田市

　ドジョウが産卵をしていました。何度か池の水が波打ちます。なんとか写ったのが上の写真です。写真の真ん中近くに二重に巻いた部分が見えます。ドジヨウのオスがメスに巻きついているのです。メスの体は左右に長く見える部分です。

　ドジヨウは15cmくらいになるといわれています。この日捕まえたドジヨウはほぼ15cmありました。ドジョウとしては大きいもののようです。全くの想像ですが、上の写真のメスの腹部が細くなっているのは産卵直後だからではないかと思います。

2012年4月29日…大阪府池田市

　2012年は7cmくらいのドジョウを4ひき確認しました。昨年は大きなものしか見られなかったので少し心配しました。
