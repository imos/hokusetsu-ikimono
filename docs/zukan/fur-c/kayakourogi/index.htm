カヤコオロギ

2014年9月18日…兵庫県三田市

　2012年の初秋、松本 勝さんにカヤコオロギの生息場所を教えていただきました。これまで北摂地域で見たことがありません。マツムシをずっと小さくしたような感じでした。ふだん出かけない場所なのでなかなか見に行くことができません。

2012年9月20日…兵庫県三田市 Photo. by M.Mat.

2014年9月18日…兵庫県三田市

　2014年、生息場所を見に連れて行ってもらいました。始め小さいので幼虫ばかりだと思っていました。ところが写真を見るとメスの産卵管が非常に長いと感じました。コオロギやキリギリスのなかまで幼虫の産卵管がこのように長い場合はありません。成虫でした。大阪府では見たことがありません。
