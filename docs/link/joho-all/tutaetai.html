メールを送ろう…ネチケットのページ

　

　このページでは、学校での授業(じゅぎょう)で、メールを送る(おくる)ときに気をつけたらいいと思うことを紹介(しょうかい)しています。ただし、授業での経験(けいけん)で書いていますので、ふつうのネチケットとは一部ちがうところもあります。一般的な(いっぱんてきな)メールを送るときのネチケットなどについては、下のリンクを見てください。

０.

　まず、先生や家の人にお願いして、『勉強でいろいろと教えてほしい』ということと、『ニックネームでメールのやりとりをさせていただいていいか』ということを、先にメールで確かめておいてもらいます。

　「ニックネームなんて失礼(しつれい)」と、思われるかも知れませんが、インターネットをするときにはいろいろな問題がおこることがありますので、始めはニックネームのほうがいいと思います。

　これまで授業では、いつもニックネームで教えてもらっていましたが、「ニックネームは、失礼だ。」と、いう人はいませんでした。ただし、途中(とちゅう)でニックネームはかえないようにしましょう。相手（あいて)の人が、だれにメールを書いていたのか分からなくなってしまうことがあります。

１.

　あいさつを書きます。

「はじめまして。」・「こんにちは。」のように、短いほうがいいでしょうね。

２.

　「ニックネーム」と「ニックネームでごめんなさい。」と書きます。

　ですから、

　はじめまして。池田市立？小学校の4年のハムちゃんといいます。ニックネームでごめんなさい。

と、なります。

３.

　ところで、同じ内容の次の二つの文章をくらべてみてください。

　はじめまして。池田市立？小学校の4年のハムちゃんといいます。ニックネームでごめんなさい。

　今、調べ学習で北海道について調べています。資料集の上空から撮った写真を見ていると、屋根の色がみんな青色でした。近所の人は、「出張のときに飛行機から見たが、他の色もあった。」と、言っていました。…

はじめまして。

池田市立？小学校の4年のハムちゃんといいます。

ニックネームでごめんなさい。

今、調べ学習で北海道について調べています。

資料集の上空から撮った写真を見ていると、屋根の色がみんな青色でした。

近所の人は、「出張のときに飛行機から見たが、他の色もあった。」と、言っていました。

…

　どちらが読みやすいでしょう。下の文章は、内容がすぐに分かります。『。』で『改行』しているからです。

　また、今は分からないでしょうが、相手の人が返事を送るときには、下のほうが書きやすくなります。

４.

　教えてほしいことを書きます。が、特にここでは、『何を』教えてほしいのかを、『具体的(ぐたいてき)に』書きます。

　以前に、平和の学習をしているときに、「今の広島はどうなっていますか。」このようなメールを子ども達が送ったことがあります。

　相手の方は、長い間かかって調べて返事をくださいました。送った子どもの気持ちも分かりますし、調べてくださった方がどれだけの時間を使ってくださったかも分かります。

　まず、本で調べたり、身近な人に話を聞いてできる限り自分で調べましょう。そして、どうしてもメールで遠くの様々な土地の人に直接教えていだかないと分からないときに、使うようにしましょう。

　ホームページやメールで調べるのも、大切な調べる方法ですが、これで自分の『調べ』ができたとは思わないでください。

　特に、人に聞く前に自分で他の方法で調べ方がないかを考えることがあなたにとってのもっと大切な勉強になります。

５.

　このようなことを説明する必要(ひつよう)はないと思いますが、相手の人に失礼(しつれい)な内容などは書きません。

６.

　メールでは、知らない人にかってなお願いをすることになりますので、『返事を待っています。はやく教えてください。』など、催促(さいそく)はしないことになっています。

７.

　学校からメールを送るときは、一度先生に内容を見てもらってくださいね。

８.

　もちろん、教えてもらったときには、お礼の返事を忘(わす)れないようにしましょう。先生にもお願いしておきましょう。

９.

　一度だけでなく、調べを続けて分からなくなったときに、また、メールを送るようになったら、ほんとうの調べ学習ができてきたと思ってください。
