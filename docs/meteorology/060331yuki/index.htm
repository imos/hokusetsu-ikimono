池田市周辺に雪が降りました

　

　2006年3月31日朝、池田市周辺の地域に雪が降りました。

　

　もうすぐ3月も終わるというのに冬のような天気です。朝7時ごろ雪が降っていました。私が見たときには屋根の上がわずかに白くなっていました。その少し前には全体にうっすらと白くなっていたということでした。あっという間にとけたようです。ちらちらとしか降っていないので小さな写真では雪のようすは分かりませんね。「こんな時期に雪が降るなんて。」と思いますが、大阪でも4月に雪が降ることはあるようです。
