クワガタムシのなかま

　

2011年6月23日…大阪府池田市五月山

　

　最近北摂地域でもクワガタムシを見る機会が減ってきました。2011年、松本　勝さんが大阪府池田市の五月山で撮影したオオクワガタの写真を見せてくださいました。めったに見ることができなくなった写真でした。なかなか見ることができなくなった自然の状態でのクワガタムシのようすを紹介します。
