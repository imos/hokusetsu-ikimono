2016年4月13日…大阪府池田市

　2016年未明、けもの道をハクビシンが歩きました。この動物もよく知られた帰化動物です。人家の周辺でも見かけることがあります。
　上の写真は目が光っていたのでその部分だけ修正しています。クリックすると大きな画像で見ることができます。

北摂の生き物

ハクビシン

(ネコ目(食肉目)ジャコウネコ科)

　全体としてどの程度分布しているのかははっきりとわかりませんが、北摂山地にはふつうに生息しているようです。
2009年1月11日…大阪府池田市

2009年1月5日…大阪府池田市

　昨年、「夜、けっこうハクビシンがうろうろしているようだ。」という話を聞きました。その場所にカメラを設置しようと思っていたら、ふだん置いているカメラにハクビシンが写りました。

2009年1月11日…大阪府池田市

　1週間ぶりにハクビシンが写りました。一枚の写真では、石の上でしっぽを上手に使ってバランスをとっていました。

2009年1月14日…大阪府池田市

　近くの別の場所でもハクビシンが写りました。後ろ姿でした。

2009年1月20日…大阪府池田市五月山

　また、ハクビシンが写りました。なんと9日前とほとんど同じ格好です…？？？

2009年3月8日…大阪府池田市五月山

　2ひきのハクビシンがカメラの近くにやってきました。しかし、光や音に驚きました。いったん姿が見えなくなってから、また、1ぴきがやってきました。でも、びっくりして逃げました。

2009年4月10日…大阪府池田市五月山

　カメラの設置場所が川沿いなのでオオアラセイトウの花が咲いています。

　ちょうどオオアラセイトウの所にハクビシンがやってきました。はじめはびっくりしたようです。左の写真です。驚きのポーズ…？

　ゆっくりと去っていったようです。

2010年1月27日…大阪府池田市五月山

　また、大阪府池田市にある五月山に自動撮影のカメラを置きました。二週続けてハクビシンが写っていました。けっこう個体数も多いのかも知れません。

2010年2月12日…大阪府池田市五月山

　ハクビシンが獣道を上っていきます。先日のものと色がよく似ているような気がします。同じ個体かも知れません。

2011年2月27日…大阪府池田市五月山

　これまでは大阪府池田市の五月山の南部でハクビシンを確認してきました。しかし、池田市の北部には生息しているのか確かめていませんでした。2011年2月27日、北部でもハクビシンが写りました。けっこう広く分布しているようです。

2011年5月12日…兵庫県三田市

　2011年5月12日、兵庫県三田市でハクビシンが写りました。ただし、大変なピンボケです。ひどい写真ですが記録としては大切かも知れませんので紹介しておきます。
 
 
2017年5月5日…大阪府池田市五月山(撮影-Mat.)
  　2017年5月5日にMat.さんが撮影されたハクビシンの写真です。夜行性でわたしは昼間に見たことがありません。
