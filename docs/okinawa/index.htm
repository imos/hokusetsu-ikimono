　沖縄本島と石垣島、竹富島、西表島、与那国島の生き物を紹介しています。2009年7月の写真をもとに、他の年のHir.さんの写真も使わせていただいています。
　哺乳類ではリュウキュウイノシシと与那国馬、イリオモテヤマネコを、爬虫類ではヤエヤマセマルハコガメ、サキシマハブ、ホオグロヤモリ、サキシマキノボリトカゲ、イシガキトカゲ、サキシマカナヘビ、キシノウエトカゲを、鳥類ではカンムリワシ、ツミ、シロハラクイナ、クジャクを、昆虫のチョウ類ではヤエヤマムラサキ、オオゴマダラ、コノハチョウ、タイワンヒメシジミ、クロマダラソテツシジミ、アオタテハモドキ、カラスアゲハ、スジグロカバマダラ、ウスイロコノマチョウ、コウトウシロシタセセリ、クロボシセセリ、シロウラナミシジミ、ツマベニチョウ、ルリボシタテハモドキ(ルリモンタテハモドキ)を、その他の昆虫ではイワサキクサゼミ、アカホシカメムシ、ナナホシキンカメムシ、イシガキモリバッタ、イリオモテモリバッタ、ヨナグニモリバッタ、ダイトウクダマキモドキ、コフキショウジョウトンボ、オオハラビロトンボ、アヤムネスジタマムシ、ワモンゴキブリを、マングローブの生き物ではヤエヤマヒルギ、オヒルギ、ミナミトビハゼ、キバウミニナ、シオマネキ類を、植物ではアダン、ハイビスカス、ヒカゲヘゴ、クワズイモ、ヤエヤマヤシ、ギンランイヌビワ、サキシマスオウ、ナリヤラン、サガリバナ、グンバイヒルガオを、海とくらしでは竹富島の家、パイナップル畑、島バナナ、ナイルテラピア、ウリミバエを紹介しています。
