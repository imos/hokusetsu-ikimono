姿を見せたのはチョウセンイタチ

2010年10月11日…大阪府池田市桃園

　2010年10月6日にチョウセンイタチの糞を玄関で見つけてから、ずっとここでくらしているのかを確かめようと、毎日家に帰ってから餌を置いてみました。これまで1日だけ朝まで餌が残っていた日がありました。しかし、残っていたその日も帰宅すると餌はなくなっていました。ネコかチョウセンイタチが食べています。11日の夜、どちらかを調べるのに自動撮影のカメラを設置しました。ところが12日の朝に見ても餌は残っていました。やはりカメラがあって近づかったのだ、と思いました。しかし、撮影枚数が多いようです。出勤の前にいそいで確かめるとチョウセンイタチが写っていました。ストロボの光かシャッター音におどろいてか、あわてまくっている様子が小さなモニターの画面からでも伝わってきました。一晩のうちに何度も挑戦していました。昼にも来るかも知れないのでカメラはそのままにして出勤しました。帰宅すると餌は残ったままでした。

　

　動物の行動を人間の勝手な解釈をしてはいけないのですが、気持ちを想像してみました。

　

「おっ、うまそうな肉だ。」

「何だ、お前は。」

「ひえーっ。」

「わぁ。またやられた。」

「これはたまらん。」

　勝手な解釈で申し訳ありません。

　かわいそうなので自動撮影は一晩でやめました。餌もくせになるといけないのでお礼に2-3日だけにしておくことにしました。
(※その後めったにないチョウセンイタチの行動を調べる機会なので観察を続けました。)

　チョウセンイタチの活動の時間帯は、翌朝の4時ごろまでは続いています。チョウセンイタチがいつもこのように活動しているのか、もう一度だけ確かめてみることにしました。

2010年10月20日…大阪府池田市桃園

　10月20日、セットをしました。ところが、午後7時5分に姿を見せたチョウセンイタチは午後7時19分には餌を持っていってしまいました。チョウセンイタチの行動の記録にはなりませんでした。自動撮影のカメラが安全なことを学習してしまったようです。

　10月22日、餌のない状態でのチョウセンイタチのようすを記録してみることにしました。

2010年10月23日…大阪府池田市桃園

　しかし、日付が変わった23日午前0時4分に姿を見せたのは目つきの悪いネコでした。

2010年10月23日…大阪府池田市桃園

　チョウセンイタチが姿を見せたのは23日の午前3時29分と4時5分でした。ネコがいると行動が変わるようです。

　餌の切り方を変えた日にネコとチョウセンイタチがやってきました。雰囲気が違うのか、朝まで餌は残りました。行動の様子がわかります。
