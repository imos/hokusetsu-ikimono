姿を消したチョウセンイタチ…？

2006年8月12日…池田市桃園

　2006年8月12日夕方、チョウセンイタチに餌をやりに行きました。

　泊をともなう出張や帰りが遅くなったとき以外は今も餌を置いています。ただし、チョウセンイタチを見なくなりました。今日は、時間をかけて観察しようと出かけました。

　犬の散歩の時間帯です。いつも餌を置く場所に行くまでに以前のラッキーに会ってしまいました。写真は、「ごちそうさま。」の表情です。ほとんどの餌は散歩中の犬のおやつになってしまいます。

　最近、チョウセンイタチの姿を見なくなったのは生態に関係があるのかも知れません。チョウセンイタチの子ども達は大きくなっていました。みんな放れていったのでしょうか…？
