池田市の木…クスノキ

　

2010年10月11日…大阪府池田市姫室町・池田市立呉服小学校

　2010年10月11日、一日延期になった校区レクリエーション大会がありました。地域や職場、職場の地域の運動会などで放送を頼まれます。この日も一日放送機器の前に座ったままでした。真正面に立派なクスノキがあります。昼食の時間帯には、大勢の人がこのクスノキの下で昼食を食べます。わたしが小学生で呉服小学校に通っていたころの記憶でも、このクスノキは大きな、幹を抱えることができないほどの大木でした。

　大阪府池田市の木はクスノキです。クスノキ科ニッケイ属の常緑高木です。「となりのトトロ」のトトロのすみ家の木がクスノキです。木から防虫剤の樟脳(しょうのう)がとれます。木の切片を衣類などの入ったものにいっしょに入れても防虫効果があると聞きます。葉にダニ室があることでも知られています。また、チョウのアオスジアゲハの主な食草です。山ではヤブニッケイなどのクスノキ科の植物を見ますが、クスノキはほとんど見ることがありません。これらのことから古い時代から人の手によって植えられたのではないかとも考えられています。また、クスノキの種子は鳥によっても運ばれ、植木鉢などでも小さな芽を出したものを見ることがあります。
